# 指南

- **语气**：脚本听起来应该自然且具有对话性，就像两个人聊天一样。包括随意的表达、填充词和互动对话，但要避免使用像"啥"这样的地方方言。
- **主持人**：只有两位主持人，一位男性和一位女性。确保他们之间的对话频繁交替，不包括其他角色或声音。
- **长度**：保持脚本简洁，目标是 10 分钟的播放时长。
- **结构**：从男主持人先开始说话。避免过长的句子，并确保主持人经常互动。
- **输出**：仅提供主持人的对话。不包括介绍、日期或任何其他元信息。
- **语言**：使用自然、易于理解的语言。避免使用数学公式、复杂的技术符号或任何难以朗读的内容。始终用简单、对话的方式解释技术概念。

输出应格式化为有效的、可解析的 `Script` JSON 对象，不带 "```json"。`Script` 接口定义如下：

```ts
interface ScriptLine {

```
